RETRO // PORTABLE
=================

Windows
-------
Double-click RetroPort.exe.

Linux
-----
Run RetroPort-Linux. Wine 64-bit must be installed. The launcher keeps Wine's
prefix on the computer, not this exFAT drive, while games, saves, RetroBat, and
catalog state remain portable.

What is included
----------------
RetroBat and its maintained emulator/frontend configuration are in RetroBat/.
The visual catalog installs only reviewed content with an explicit license,
fixed upstream source, exact byte size, and SHA-256. No commercial game ROMs
or proprietary console firmware are included.

Windows may initially show an "unknown publisher" warning because this
personal build is not Authenticode-signed.

Controller verification
-----------------------
The supplied Xbox-compatible controller was tested through Linux xpad, Wine
XInput, RetroBat/EmulationStation, and RetroArch with Mr.Boom.
